## Development

Make sure you have [Yarn](https://yarnpkg.com) installed.

Start Dev Server:

```sh
yarn docs:dev
```

Build:

```sh
yarn docs:build
```

## Deploy to GitHub Pages Manually

```sh
./deploy.sh
```
